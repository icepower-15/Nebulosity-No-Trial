# Nebulosity

## Overview

Nebulosity is software designed for image capture and processing tailored to Windows users. It facilitates the acquisition of astrophotography data through a straightforward interface, enabling users to manage and process images effectively. The project aims to support Windows-based workflows in astronomical imaging.

## Key Features

**Support for various camera types including CCD and CMOS**  
**Live image preview during capture**  
**Automated flat frame acquisition**  
**Basic image processing tools such as stacking and alignment**  
**Integration with common astronomical hardware**  
**Configurable capture sequences for unattended operation**  
**Detailed logging of capture sessions**  
**User-friendly interface optimized for Windows environments**

## Why Nebulosity?

Nebulosity is developed to provide a reliable and clear workflow for astrophotography image capture on Windows systems. Its design focuses on usability and consistent performance, offering a practical toolset for managing image acquisition without unnecessary complexity. The software balances functionality with straightforward controls to support diverse imaging needs.

## Benefits

Nebulosity allows users to handle image capture tasks with precision and control. It streamlines workflows related to astrophotography by combining capture and preliminary processing features. The software is structured to minimize operational interruptions and facilitate efficient data acquisition for Windows users.

## Compatibility

This repository targets Windows operating systems, ensuring stable performance and efficient operation within this platform. Nebulosity is tailored to leverage Windows system capabilities for reliable image capture and processing.
